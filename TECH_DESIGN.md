# 🚀 科技风格首页设计

## 🎨 设计理念

我们重新设计了首页，采用现代科技风格，创造沉浸式的游戏体验。

### 🌟 设计特色

- **深空黑背景** - `#000015` 营造深邃科技感
- **霓虹绿主色** - `#00ff88` 科技感十足的荧光绿
- **极简布局** - 去除冗余信息，专注核心功能
- **等宽字体** - Courier字体增强编程感
- **发光效果** - 阴影和边框营造未来感

## 🎯 界面元素

### 标题区域
- "BOARD GAMES" 标签 - 带边框的科技标识
- 主标题 - 等宽字体，字母间距增加
- "SELECT GAME" 副标题 - 英文提示增强科技感

### 游戏卡片
- **透明背景** - `rgba(255, 255, 255, 0.05)` 半透明效果
- **彩色边框** - 每个游戏对应不同颜色
- **发光图标** - 圆形容器包含游戏emoji
- **状态指示** - 可用游戏显示发光点，未来游戏显示"SOON"标识

### 颜色方案
```typescript
井字棋: '#00ff88'  // 霓虹绿
跳棋:   '#ff0080'  // 霓虹粉
象棋:   '#0080ff'  // 霓虹蓝  
五子棋: '#ff8000'  // 霓虹橙
```
